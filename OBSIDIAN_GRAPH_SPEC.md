# Obsidian 关系图谱复刻规格

> 文档状态：初稿，等待人工校订  
> 整理日期：2026-06-12  
> 主要来源：[Obsidian Graph view 官方帮助](https://obsidian.md/help/plugins/graph)

## 0. 最低确认项

不需要逐项填写本文档中的尺寸、颜色和物理参数。除以下项目外，其余内容均由
实现侧按照 Obsidian 默认主题、官方行为和浏览器环境自行决定。

### 必须由用户确认

1. **分类功能如何处理**
   - [ x ] 严格保持 Obsidian 一致：图谱中不提供分类关系。
   - [ ] 作为 VuePress 扩展保留：默认关闭，可在高级设置中开启。

2. **附件节点的范围**
   - [  ] 常用附件：图片、PDF、音视频。
   - [ x ] 所有被 Markdown 引用的本地非 Markdown 文件。
   - [ ] 不实现附件节点。

3. **未创建文件节点**
   - [ x ] 实现：无法解析的内部链接显示为半透明节点。
   - [ ] 不实现：继续忽略无法解析的链接。

### 已根据当前校订内容确定

- 节点单击后打开页面。
- 按下并移动指针时拖动节点，不触发页面跳转。
- 不存在选中节点状态。
- 不存在当前页面节点专用颜色。
- 箭头默认关闭。
- 双向链接显示为一条视觉边。
- 标签显示为独立节点，不通过笔记颜色表达。
- 普通笔记默认使用统一颜色。
- 悬停时高亮一跳关系并弱化其他内容。
- 浅色和深色主题都需要保持可读。
- 不实现完整 Obsidian 搜索语法、右键菜单和时间动画。

### 由实现侧自行决定

- 具体颜色值：优先读取 VuePress 主题 CSS 变量，并提供可靠的浅色、深色回退值。
- 节点尺寸公式：使用入链数为主要权重，设置合理的最小值和最大值。
- 文字显示阈值：结合缩放级别和节点重要程度动态显示。
- 字体、字号、描边、标签间距和视觉过渡。
- 画布背景、内边距、自动居中和初始缩放。
- 悬停弱化透明度和动画时长。
- 缩放范围、鼠标中心缩放、键盘步长和缓动。
- 分组优先级：列表中靠前的分组优先。
- 空分组不生效；删除分组不弹出确认。
- 力学滑块范围、默认值和参数映射。
- 力学设置实时生效。
- 全局图谱和局部图谱使用独立视图状态，共享显示与分组设置。
- 恢复默认设置会清空自定义分组。
- 局部图谱深度范围使用 1–5。
- 搜索只显示直接匹配节点，不自动扩展一跳邻居。
- 移动端和侧边栏采用适合网页的响应式实现。

## 1. 目标

本文档描述 Obsidian 全局图谱和局部图谱的功能、交互与视觉行为，作为
`vuepress-plugin-knowledge-graph` 后续实现和验收依据。

目标是在 VuePress 和浏览器能力允许的范围内，使图谱的使用体验尽量与
Obsidian 一致。

明确不纳入本次复刻范围：

- Obsidian 完整搜索语法
- 节点右键菜单
- 按文件创建时间播放的图谱动画

## 2. 信息可信度

本文档使用以下标记：

- **官方确认**：Obsidian 官方帮助明确说明。
- **当前推定**：根据常见 Obsidian 使用体验整理，需要人工复核。
- **待补充**：官方帮助没有精确说明，需要通过截图或实际操作确认。
- **项目决策**：VuePress 版本需要自行确定的行为。

## 3. 基本数据模型

### 3.1 节点

**官方确认**

- 圆形节点代表笔记。
- 被更多节点引用的笔记显示得更大。
- 开启标签显示后，标签会作为独立节点出现在图谱中。
- 开启附件显示后，附件会作为独立节点出现。
- 可选择显示尚未创建、但已被链接的文件。
- 可选择显示没有任何链接的孤立文件。

### 3.2 连线

**官方确认**

- 连线代表笔记之间的内部链接。
- 可显示箭头，用于表达链接方向。（默认不显示箭头）

**待补充**

- 双向链接是否绘制为两条重叠边，还是一条无向视觉边。（一条无向视觉边）
- 标签节点与笔记之间连线的默认颜色和透明度。（你看着来，只要能看的清楚就可以，注意主题切换的情况）
- 未创建文件、附件节点对应连线的视觉差异。（这个不用视觉差异）

### 3.3 VuePress 数据映射建议

| Obsidian 概念 | VuePress 数据 |
| --- | --- |
| 笔记节点 | Markdown 页面 |
| 内部链接 | Wiki Link 和 Markdown 相对链接 |
| 标签节点 | Frontmatter `tags` / `tag` |
| 附件节点 | Markdown 引用的图片、PDF 等静态资源 |
| 未创建文件 | 无法解析到现有 Markdown 的内部链接目标 | 
| 孤立节点 | 没有任何内部链接、标签边或附件边的页面 |

`categories` 不是 Obsidian 图谱的原生概念。为保持一致，建议默认不显示分类
关系，将它作为 VuePress 扩展功能保留。

## 4. 默认视觉

### 4.1 节点颜色

**官方确认**

- 分组可以为符合条件的节点指定颜色。

**当前推定**

- 未命中任何分组的普通笔记使用统一的主题节点颜色。
- 标签不会通过改变笔记节点颜色来表达，而是显示为独立标签节点。
- 悬停、选中、当前页面和弱化状态会覆盖普通节点颜色。

**待补充**

obsidian没有选中节点，没有当前页面节点的概念，单击即可进入页面，单击不放下可以拖拽。

| 元素 | 浅色主题 | 深色主题 |
| --- | --- | --- |
| 普通笔记节点 | `白` | `深蓝` |
| 标签节点 | `红` | `红` |
| 附件节点 | `主题色` | `主题色` |
| 未创建文件节点 | `更透明的灰色` | `更透明的灰色` |
| 当前页面节点 | 不适用 | 不适用 |
| 悬停节点 | `主题色` | `主题色` |
| 选中节点 | 不适用 | 不适用 |
| 非关联节点 | `原色变暗` | `原色变暗` |
| 普通连线 | `白` | `灰` |
| 高亮连线 | `主题色` | `主题色` |

### 4.2 节点大小

**官方确认**

- 被更多节点引用的笔记节点更大。
- “Node size”滑块控制节点整体大小。

**待补充**

- 节点半径与入链数之间的具体计算方式。
- 是否只计算入链，还是同时计算入链和出链。
- 节点半径的最小值、最大值。
- 标签、附件和未创建文件是否使用不同基础大小。

### 4.3 标签文字

**官方确认**

- “Text fade threshold”控制笔记名称文字的淡入淡出阈值。

**当前推定**

- 缩放较远时隐藏大部分标签。
- 缩放靠近时逐渐显示更多标签。
- 悬停节点和当前节点的文字应始终可见。

**待补充**

- 文字出现和消失的缩放阈值。
- 是否根据节点重要程度优先显示文字。
- 字体、字号、字重、文字与节点的间距。
- 文字是否有描边、阴影或背景遮罩。

### 4.4 画布

**待补充**

- 浅色和深色主题背景颜色。
- 图谱是否完全占满内容区域。
- 设置面板打开后是覆盖画布还是压缩画布。
- 画布边缘是否存在内边距。
- 初次打开时的默认缩放比例和居中方式。

## 5. 基础交互

### 5.1 打开笔记

**官方确认**

- 单击节点打开对应笔记。

**项目冲突**

当前 VuePress 插件采用：

- 单击：选中并居中节点。
- 双击：打开页面。

最终采用 Obsidian 行为：

- 单击节点打开页面。
- 指针按下后发生拖动时只移动节点，不打开页面。

### 5.2 悬停

**官方确认**

- 悬停节点会高亮该节点的连接。

**待补充**

- 一跳关联节点是否同时高亮。
- 非关联节点和连线的目标透明度。
- 高亮切换是否有过渡动画。

### 5.3 拖动

**待补充**

- 节点是否可以拖动。
- 松开节点后是否固定，还是重新参与力导向布局。
- 是否存在解除固定节点的操作。
- 拖动画布和拖动节点时的光标样式。

### 5.4 缩放和平移

**官方确认**

- 鼠标滚轮缩放。
- 鼠标拖动画布进行平移。
- `+`、`-` 键缩放。
- 方向键平移。
- 按住 Shift 配合方向键可加速平移。

**待补充**

- 最小和最大缩放级别。
- 缩放是否以鼠标位置为中心。
- 键盘每次缩放和平移的距离。
- 缩放和平移的缓动时间。

## 6. 设置面板

**官方确认**

- 设置按钮位于图谱右上角。
- 可恢复默认设置。
- 设置分为 Filters、Groups、Display 和 Forces。

### 6.1 Filters

| 设置 | Obsidian 行为 | 本项目目标 |
| --- | --- | --- |
| Search files | 按搜索条件过滤文件 | 支持普通关键词匹配 |
| Tags | 显示或隐藏标签节点 | 需要改为独立标签节点 |
| Attachments | 显示或隐藏附件节点 | 待实现 |
| Existing files only | 隐藏未创建文件节点 | 待实现 |
| Orphans | 显示或隐藏孤立节点 | 已有近似实现 |

普通关键词应至少匹配：

- 标题
- 文件路径
- 标签
- 分类扩展字段

搜索结果只显示直接匹配节点，不自动扩展一跳邻居。

### 6.2 Groups

**官方确认**

- 可以创建多个颜色分组。
- 每个分组包含搜索条件和颜色。

**待补充**

- 一个节点同时命中多个分组时的优先级。
- 分组排列是否可以拖动排序。
- 空条件分组如何处理。
- 默认分组颜色。
- 删除分组是否需要确认。

本项目不实现完整 Obsidian 搜索语法，但分组条件至少匹配标题、路径、标签和
分类。

### 6.3 Display

| 设置 | 官方含义 | 实现状态 |
| --- | --- | --- |
| Arrows | 显示链接方向 | 已实现 |
| Text fade threshold | 调整文字淡入阈值 | 当前是透明度滑块，需要改造 |
| Node size | 调整节点大小 | 已实现近似版本 |
| Link thickness | 调整连线粗细 | 已实现 |
| Animate | 时间动画 | 明确不实现 |

### 6.4 Forces

| 设置 | 官方含义 | 实现状态 |
| --- | --- | --- |
| Center force | 控制图谱紧凑程度和圆形聚合 | 已实现近似版本 |
| Repel force | 控制节点之间的排斥力 | 已实现 |
| Link force | 控制连线拉力 | 已实现 |
| Link distance | 控制节点间连线长度 | 已实现 |

**待补充**

- 每个滑块的最小值、最大值、默认值。
- 滑块是否线性映射到物理参数。
- 拖动滑块时是否实时重新布局。
- 恢复默认设置后的准确参数。

## 7. 全局图谱

**官方确认**

- 全局图谱显示库中的全部可见文件及关系。
- 过滤设置决定最终显示的节点。

**目标行为**

- 初次进入时自动适配全部节点。
- 修改过滤条件后重新计算布局并适配可见节点。
- 设置面板状态和图谱参数持久化。

**待补充**

- Obsidian 是否持久化当前缩放和平移位置。
- 重新打开图谱时是否复用上次节点坐标。
- 全局图谱和局部图谱是否使用独立设置。

## 8. 局部图谱

**官方确认**

- 局部图谱以当前活动笔记为中心。
- 显示与活动笔记相连接的节点。
- 支持全局图谱的全部设置。
- 额外支持关系深度设置。
- 每增加一层深度，会继续显示上一层节点所连接的节点。

**目标行为**

- 当前页面节点始终位于局部图谱中心附近。
- 当前页面节点应有明确但符合主题的视觉强调。
- 页面切换后自动更新局部图谱中心。
- 深度支持范围：1–5。

**待补充**

- 深度滑块的实际最小值和最大值。
- 局部图谱是否显示标签和附件节点。
- 当前页面改变时是否保留缩放位置。
- 局部图谱默认中心力和全局图谱是否不同。

## 9. 设置持久化

**当前推定**

- 图谱设置应在刷新页面后保留。
- 全局图谱和局部图谱可能拥有独立配置。

**待补充**

- Obsidian 中哪些设置是全局共享的。
- 哪些设置按工作区或视图实例保存。
- “Restore default settings”是否同时清空分组。

## 10. 响应式与侧边栏

Obsidian 桌面应用没有与 VuePress 完全相同的移动端和网页侧边栏约束，因此以下
属于项目决策：

- 小屏幕下设置面板占满图谱宽度。
- 图谱侧边栏可以调整宽度。
- 全屏模式退出后恢复原侧边栏状态。
- 图谱不能遮挡站点必要的导航和关闭按钮。

## 11. 当前插件与 Obsidian 的主要差异

| 项目 | 当前插件 | Obsidian | 后续动作 |
| --- | --- | --- | --- |
| 打开页面 | 双击节点 | 单击节点 | 等待确认 |
| 标签 | 相同标签的页面直接连边 | 标签作为独立节点 | 需要重构 |
| 分类 | 支持分类关系 | 无原生分类概念 | 默认隐藏或作为扩展 |
| 附件 | 不生成节点 | 可显示附件节点 | 待实现 |
| 未创建文件 | 忽略 | 可显示并可过滤 | 待实现 |
| 搜索 | 普通包含匹配并扩展邻居 | Obsidian 搜索条件 | 保留简化语法，确认结果范围 |
| 文字设置 | 直接控制透明度 | 控制淡入阈值 | 需要重构 |
| 节点拖动 | 拖动后固定 | 待实测 | 等待补充 |
| 键盘导航 | 未实现 | 支持 | 待实现 |
| 全屏 | 浏览器全屏 | 应用视图 | 保留 Web 版本 |
